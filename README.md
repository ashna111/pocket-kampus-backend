## Features

- Calendar View
- Reminders
- Subjects : Submissions and Lectures
- Attendance Tracker
- Events Explore Page
- Email upon Event Registration
- e-Library

---

## Requirements

- Node.js and Node package manager (npm)
- MongoDB (with path set correctly)

---

## Installation & Setup

```sh
$ git clone https://github.com/RyanDsilva/college-kampus-backend.git

$ mongod

$ node app.js
```
